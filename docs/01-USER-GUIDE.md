# SWTC 3D 列印機預約 + 庫存管理系統 — 使用說明

> **版本**：v2.0（Cloud Function 架構）
> **更新日期**：2026/06/18
> **負責人**：jiliao

---

## 一、系統概覽

本系統是公司內部 3D 列印機（Formlabs Form4 / Form4L 等）的預約管理與材料庫存追蹤工具。

### 主要功能

| 模組 | 網址 | 主要功能 |
|------|------|---------|
| 預約系統 | `3DP-BK.html` | 機台預約、查詢、甘特圖、即時狀態 |
| 庫存管理 | `inventory.html` | 樹脂罐、備料庫存、消耗紀錄、月度分析 |

兩個系統共用同一個帳號（Firebase Auth 登入），即時資料來自 Firestore 雲端資料庫。

---

## 二、使用者角色

| 角色 | 預約系統 | 庫存系統 |
|------|---------|---------|
| **viewer** | 查看所有預約、機台狀態 | 查看庫存、消耗紀錄 |
| **editor** | viewer 權限 + 新增/編輯/刪除預約 | viewer 權限 + 編輯庫存、新增入庫 |
| **admin** | editor 權限 | editor 權限 + 觸發同步、清除重建、設定安全庫存、去除重複 |

新使用者建立後預設為 viewer，admin 可至 Firestore `users/{uid}` 修改 `role` 欄位。

---

## 三、預約系統使用方式

### 登入

1. 打開 `3DP-BK.html`
2. 輸入 email + 密碼登入

### 新增預約（editor / admin）

1. 點頂部「+ 新增預約」按鈕
2. 填入：
   - **日期**（必填）
   - **業務（Sales）**
   - **機台**（Form4 / Form4L / MarkTwo / UR / Other）
   - **客戶 / 案號 / 備註**
   - **是否已下單**
   - **目的（量產 / 樣品 / 內部）**
   - **負責工程師**
   - **狀態**（待確認 / 執行中 / 已完成 / 異常·取消）
3. 點「儲存」

### 查看預約

- **表格模式**：列表，可排序、過濾，每頁 10 筆
- **甘特圖模式**：預設顯示，按機台分線顯示時間軸

### 即時機台狀態

頁面下方有列印機即時狀態卡片，顯示：
- 機台名稱 + 別名（Form4 / AluminumBowfin 等）
- 當前 status（IDLE / PRINTING / ERROR 等）
- 機台上裝著的樹脂罐材料與剩餘量

**資料來自 Cloud Function 每 10 分鐘自動同步**，前端 onSnapshot 即時推送，**不需手動重新整理**。

## 四、庫存系統使用方式

### 四個分頁

| 分頁 | 內容 |
|------|------|
| **庫存總覽** | 月度消耗統計、低庫存警告、機台樹脂罐狀態 |
| **材料庫存** | 備料庫存清單、機台樹脂罐（摺疊）、可編輯 |
| **消耗紀錄** | 列印消耗、入庫、調整的歷史，預設 30 天 |
| **月度分析** | 圓餅圖、月度趨勢圖、詳細統計表 |

### 材料庫存頁

#### 欄位說明

| 欄位 | 來源 | 是否可編輯 |
|------|------|-----------|
| **材料名稱** | 系統內建 + 自動偵測新材料 | 否 |
| **庫存 (L)** | `inv.stock[material].total_ml / 1000` | ✓ editor 編輯模式 |
| **機台上 (L)** | Cloud Function 從 Formlabs API 同步 | 否（自動更新） |
| **安全庫存 (L)** | 手動設定 | ✓ admin only |
| **狀態** | 自動計算：庫存 < 安全庫存 → 需叫料 | 否 |
| **備註** | 手動輸入 | ✓ editor 編輯模式 |
| **最後更新** | 系統自動記錄 | 否 |
| **操作** | 刪除按鈕 | ✓ editor 編輯模式關閉時 |

#### 整頁編輯模式（editor / admin）

1. 點右上「✏️ 編輯」按鈕
2. 所有列的「庫存 (L)」和「備註」input 啟用（藍框）
3. 編輯多列數值
4. 點「✓ 儲存全部」→ **一次性**寫入 Firestore（節省成本）
5. 或點「✕ 取消」→ 還原為儲存前的值

> 編輯模式中其他列的刪除鍵會 disabled，避免誤操作

#### 新增入庫

點右上「📦 新增入庫」→ 選材料 + 數量 → 儲存。會自動寫入一筆 history（type=stockin）。

#### 機台樹脂罐（摺疊區）

材料庫存頁上方有「機台樹脂罐」折疊區，預設收起。展開可看：
- 每台追蹤機台（Form4 / Form4L）的樹脂罐
- 卡片邊框顏色依機台狀態：
  - 🟢 綠：閒置
  - 🔵 藍：列印中
  - 🔴 紅：異常
  - ⚪ 灰：未知 / 未同步

數值會隨 Cloud Function 同步自動更新。

### 消耗紀錄頁

#### 預設範圍

只訂閱**最近 30 天**資料（節省 Firestore 讀取成本）。頁面頂部顯示：
```
資料範圍：5/19 ~ 6/18（30 天，共 XX 筆）
```

#### 載入更早資料

點「⏮ 載入更早一個月」按鈕：
- 每次往前推 30 天
- 使用 `getDocs` 一次性讀取（非訂閱）
- F5 重整後**回到預設 30 天**（不會永久擴展訂閱範圍）

#### 紀錄類型

| Type | 顯示 | 說明 |
|------|------|------|
| `consume` | 🔴 列印消耗 | 列印完成 / 列印錯誤 |
| `aborted` | 🟠 列印中止 | 列印被取消 |
| `stockin` | 🟢 備料入庫 | 手動新增入庫 |
| `manual` | ⚙️ 手動調整 | 編輯庫存的差異紀錄 |

#### 篩選 + 排序

- 材料 / 機台 / 類型 / 日期區間
- 按欄位點 header 排序

#### 匯出 Excel

點右上「📤 匯出」→ 匯出當前篩選結果為 xlsx。

### 月度分析頁

- **圓餅圖**：本年度各材料消耗佔比
- **月度趨勢圖**：折線 / 長條切換，顯示每月消耗
- **詳細統計**：摺疊區，每月每材料數值

### 管理員專屬功能（admin only）

| 按鈕 | 用途 |
|------|------|
| **🔄 觸發同步** | 立即執行 Cloud Function 同步（一般每 10 分鐘自動） |
| **🗑 清除並重建** | 清空所有 history + last_processed_prints，需手動跑 backfill |
| **🚫 去除重複** | 掃描 history 按 print_guid 去重，刪除多餘紀錄 |
| **⚙️ 安全庫存設定** | 設定每個材料的低庫存警告閾值 |
| **🚫 隱藏材料設定** | 把舊版本材料從清單隱藏（黑名單） |

## 五、使用情境

### 情境 A：列印機完成一筆 print

1. Formlabs 機台列印完成
2. **10 分鐘內** Cloud Function 自動偵測
3. 寫入：
   - Firestore `printer_status/current`（機台狀態 + cartridge 剩餘量）
   - Firestore `inventory_history/{print_guid}`（消耗紀錄）
   - Firestore `inventory/main.cartridges`（機台樹脂罐更新）
4. 前端 onSnapshot 即時收到變更
5. 使用者看到：
   - 庫存頁「機台上 (L)」減少
   - 消耗紀錄頁出現新一筆 `列印消耗`
   - 月度分析自動更新

### 情境 B：使用者手動換樹脂罐

1. 使用者拿備料一罐裝到機台上
2. Formlabs API 偵測到（cartridge serial 變更）
3. **10 分鐘內** Cloud Function 同步
4. 「機台上 (L)」更新為新罐容量（接近 1.0 L）
5. **備料庫存「庫存 (L)」不會自動扣**

⚠️ **使用者需手動編輯庫存**：到材料庫存頁 → 編輯 → 該材料庫存減少 1 L → 儲存。

理由：無法 100% 確定使用者拿的是備料、客戶送的、實驗用的還是別處轉用的，需人工確認。

### 情境 C：admin 設定安全庫存

1. 進材料庫存頁
2. 點上方「⚙️ 安全庫存設定」（admin 才看得到）
3. 表格中設定每個材料的閾值（L）
4. 儲存
5. 之後庫存 < 閾值的材料會顯示「需叫料」紅色標籤

### 情境 D：發現消耗紀錄有重複

1. admin 進消耗紀錄頁
2. 點上方「🚫 去除重複」
3. 系統掃描所有 history 按 print_guid 去重
4. 保留每組第一筆，刪除其他
5. 出現「✓ 已刪除 N 筆重複紀錄」

## 六、即時資料對應表

| 看到的數值 | 來源 | 更新頻率 |
|----------|------|---------|
| 預約清單 | Firestore `bookings` | 即時 (onSnapshot) |
| 機台即時狀態 | Firestore `printer_status/current` | Cloud Function 10 分鐘自動寫入 → 即時推送 |
| 機台上樹脂罐剩餘 | Firestore `inventory/main.cartridges` | Cloud Function 10 分鐘自動寫入 → 即時推送 |
| 備料庫存 (L) | Firestore `inventory/main.stock` | 使用者手動編輯 → 即時 |
| 安全庫存 | Firestore `inventory/main.safety` | admin 手動 |
| 消耗紀錄 | Firestore `inventory_history`（最近 30 天） | Cloud Function 10 分鐘自動寫入 → 即時推送 |
